# Contributing to TradeFlow Ledger

Thank you for your interest in contributing! TradeFlow is an open-source project
built on the XRP Ledger to modernize trade finance for importers and exporters globally.

-----

## Ways to Contribute

- **Report bugs** — Open an Issue describing what went wrong
- **Suggest features** — Open an Issue with your idea
- **Submit code** — Fork the repo, make your changes, open a Pull Request
- **Improve docs** — Fix typos, clarify steps, add examples

-----

## Getting Started

1. Fork this repository
1. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Type-tradeflow-poc.git`
1. Install dependencies: `npm install`
1. Create a branch: `git checkout -b feature/your-feature-name`
1. Make your changes
1. Test on XRPL testnet before submitting
1. Open a Pull Request with a clear description of what you changed and why

-----

## Code Guidelines

- Use clear, descriptive variable names
- Add comments explaining *why*, not just *what*
- Test all XRPL transactions on testnet first — never commit mainnet seeds or private keys
- Keep commits focused — one feature or fix per commit

-----

## Security

**Never commit wallet seeds, private keys, or API keys.** If you accidentally do,
rotate your keys immediately and let us know by opening an Issue.

-----

## Questions?

Open an Issue and we'll get back to you.
